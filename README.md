# Undangan Pernikahan Achmad & Suci

## Menjalankan

Buka `index.html` langsung melalui browser, atau jalankan server lokal:

```bash
python3 -m http.server 5500
```

Lalu buka `http://localhost:5500`.

## Nama tamu melalui URL

Tambahkan parameter `to`:

```text
http://localhost:5500/?to=Nama%20Tamu
```
